What is the Document Object Model (DOM)?

The Document Object Model (DOM) is a programming interface for web documents. It represents the structure of a document as a tree of objects, allowing programs to interact with and manipulate the document’s content, structure, and style.

How do you select an element from the DOM?

To select an element from the DOM, you can use methods like:

getElementById: Selects an element by its unique ID.

document.getElementById("elementId")

querySelector: Selects the first element that matches a CSS selector.

document.querySelector(".className");

querySelectorAll: Selects all elements that match a CSS selector.

document.querySelectorAll("div");
What is event delegation in the context of the DOM, and why is it useful?

Event delegation is a technique in the DOM where a single event listener is added to a parent element instead of multiple listeners to individual child elements. This leverages event bubbling, where events propagate up the DOM tree

How do you manipulate an element's attributes and styles using the DOM?

To manipulate an element’s attributes and styles using the DOM:

Attributes: Use setAttribute and getAttribute.

element.setAttribute("attributeName", "value");
let value = element.getAttribute("attributeName");

Styles: Use the style property.

element.style.propertyName = "value";

For example, to change the src attribute of an image and its width:

> let img = document.querySelector("img");
> img.setAttribute("src", "newImage.jpg");
> img.style.width = "200px";
